## Descripción General
Este proyecto es una **tarjeta de vista previa de artículo responsive** que se adapta a diferentes tamaños de pantalla.  
Incluye un ícono de compartir que muestra enlaces a redes sociales al hacer clic, agregando un pequeño comportamiento interactivo al componente.

## El Desafío
Los usuarios deberían poder:

- Ver el diseño óptimo según el tamaño de pantalla de su dispositivo.  
- Ver los enlaces de redes sociales al hacer clic en el ícono de compartir.

## Mi Proceso
- Usé Flexbox para estructurar y alinear los elementos.  
- Utilicé propiedades personalizadas de CSS para mantener los colores consistentes.  
- Agregué lógica sencilla en JavaScript para mostrar y ocultar el panel de compartir.  
- Usé `position: absolute` para colocar correctamente el tooltip de compartir sobre el botón.

## Tecnologías Usadas
- **HTML5**  
- **CSS3**  
- **Flexbox**  
- **Propiedades personalizadas de CSS**  
- **JavaScript Vanilla**

## Lo Que Aprendí
- Cómo **alternar la visibilidad de elementos dinámicamente** usando JavaScript.  
- Cómo **posicionar popups o tooltips** relativos a otros elementos.  
- Cómo **gestionar paletas de colores** usando variables CSS.
